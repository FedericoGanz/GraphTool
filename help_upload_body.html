General

  • Data: country-year panel.
  • File format: .csv

Variables that must be included

  1. 'Var_name': variable name.
  2. 'Units': units in which the variable is expressed.
  3. 'Ctry_iso': country ISO 3 code.
  4. 'Year': year (non-numeric values will be treated as missing).
  5. 'Value': value (non-numeric values will be treated as missing).

Additional variables that can be included

  1. 'Database': source of the data. If not included then the file name is used.
  2. 'Var_code': variable code. It should be unique for each variable. If not included, then 'Var_name'_'Database' is used.
  3. 'Description' (Metadata): variable description.
  4. 'Source_data' (Metadata): database where the information was retrieved.
  5. 'Source_org' (Metadata): organization that collects the data.

When loading, data will be filtered using the country selection and year range parameters.
